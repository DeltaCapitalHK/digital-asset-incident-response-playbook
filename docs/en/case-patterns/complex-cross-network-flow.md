# Complex Cross-Network Asset Flow Pattern

This page describes incidents where stolen or scam-related funds move through multiple conversion and cross-network layers.

## Anonymized Case Summary

In one anonymized fake-arbitrage-platform scenario, a victim transferred approximately 108,743 USDT to an EVM-chain address. Within hours, the funds moved through multiple downstream layers before signs of possible platform-facing activity appeared.

The main difficulty is that the path is fragmented across multiple networks and transaction types. The documentation task is to produce a clear, reviewable summary without exposing internal analysis methods.

The public case pattern can be summarized at a high level. Detailed route interpretation, linkage logic, and node-attribution methods are not published:

```text
Victim wallet
  -> EVM suspicious recipient
  -> multi-layer downstream movement
  -> later-path review
  -> suspected platform or other reviewable node
```

Real addresses, transaction hashes, exchange accounts, internal review details, and personal identifiers are not published.

## Typical Scenario

Funds are transferred to a source-chain address, then:

1. Moved through multiple conversion layers
2. Moved across one or more network environments
3. Consolidated or approached a platform-controlled environment

## Materials to Prepare

- First loss transaction hash
- Source-chain victim and recipient addresses
- Key transaction records across the visible path
- Later-path recipient address, if visible
- Key later-path transaction records
- Consolidation or suspected exchange address
- Chat records, platform screenshots, and source-of-funds materials

## Public Method Boundary

This public page intentionally does not disclose internal tracing logic, attribution criteria, review thresholds, or platform-entry assessment methods. It only explains the case type and the categories of materials that may need to be organized.

## Public Material Categories

```text
first-loss transaction and incident narrative
source-chain path summary
cross-network movement summary
later-path summary
source-of-funds and ownership materials
reporting exchange or issuer contact materials
```

## Important Notice

Complex flow review should be written in evidence-chain language: what is confirmed, what is suspected, and what requires further verification by exchanges, law enforcement, or legal process.
