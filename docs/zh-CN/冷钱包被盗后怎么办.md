# 冷钱包被盗后怎么办

冷钱包被盗通常不是“冷钱包硬件自然失效”，而是助记词、私钥、备份、签名环境或供应链环节出现问题。

## 常见原因

- 助记词拍照、截图或上传云盘
- 助记词输入到假网站、假客服页面或假钱包 App
- 购买到被篡改的硬件钱包
- 使用不安全电脑或手机连接冷钱包
- 签署恶意交易或授权
- 恢复钱包时使用了不可信设备
- 备份纸、金属板或保管地点被他人接触

## 立刻处理

1. 不要继续使用原助记词派生的钱包。
2. 使用可信渠道和干净设备创建新钱包。
3. 转移尚未被盗的资产。
4. 保存硬件钱包型号、购买渠道、初始化过程、固件版本等信息。
5. 保存被盗交易哈希和地址。
6. 检查是否是授权被盗，而非助记词完全泄露。

## 需要整理的材料

- 冷钱包品牌和型号
- 购买渠道和时间
- 是否自己首次初始化
- 助记词是否拍照、上传、复制、输入过网页
- 是否连接过异常网站或 DApp
- 被盗交易哈希
- 被盗资产列表
- 剩余资产处理记录

## 注意事项

如果助记词已经泄露，原钱包不再安全。后续重点是保护剩余资产、保存证据、整理链上路径，并根据资金去向准备交易所、发行方、执法或法律程序所需材料。
