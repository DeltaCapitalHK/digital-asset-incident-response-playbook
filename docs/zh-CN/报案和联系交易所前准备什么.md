# 报案和联系交易所前准备什么

报案或联系交易所前，先把材料整理成清晰的事实包，可以减少重复沟通和信息遗漏。

## 报案前准备

- 身份信息和联系方式
- 事件经过说明
- 每笔损失交易哈希
- 钱包地址和收款地址
- 虚假平台网址或 App 信息
- 对方联系方式和聊天记录
- 转账截图和区块浏览器链接
- 资金来源记录
- 损失金额汇总

## 联系交易所前准备

如果资金疑似进入交易所，需要准备：

- 疑似交易所充值地址
- 进入交易所前的完整资金路径
- 每一跳交易哈希
- 资金与受害钱包的关联说明
- 受害人身份和联系方式
- 报案回执，如已有
- 需要交易所协助的事项：冻结账户、保全信息、协助执法调取等

## 事件说明建议结构

```text
标题：关于 [日期] 数字资产诈骗/盗取事件的情况说明

1. 受害人基本情况
2. 事件发生经过
3. 损失资产和交易哈希
4. 链上资金流向
5. 已保存证据
6. 已采取措施
7. 请求协助事项
```

## 注意事项

不要在材料中夸大、伪造或猜测事实。能证明的写事实，不能证明的写“疑似”或“待核实”。
