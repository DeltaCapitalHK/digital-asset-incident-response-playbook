# 联系交易所冻结资金需要什么

如果怀疑被骗或被盗资金进入某个交易所，联系交易所前应先准备清楚的交易路径、受害证明、资金来源、报案材料和请求事项。交易所是否冻结或协查，取决于平台规则、证据质量和合规要求。

## 先确认

- 是否有明确的交易所充值线索。
- 是否只是疑似进入交易所，还是已有平台可核实地址。
- 是否有完整的首笔损失交易和后续路径摘要。
- 是否已有报案回执、案件编号或执法机关联系信息。
- 是否能证明资金属于受害人。

## 材料清单

| 材料 | 说明 |
|---|---|
| 受害人信息 | 姓名、联系方式、身份证明，按平台要求提供 |
| 事件经过 | 诈骗或被盗的时间线 |
| 首笔损失交易 | 哈希、地址、网络、金额、时间 |
| 后续路径摘要 | 关键交易记录和可见流向 |
| 疑似平台节点 | 需要平台协助核实的地址或交易 |
| 资金来源 | 买币、提现、银行记录、历史持仓等 |
| 报案材料 | 报案回执、案件编号，如已有 |
| 请求事项 | 请求冻结、保存记录、协助核实或提交执法流程 |

## 邮件或工单建议结构

```text
Subject: Request for review and preservation of suspected stolen digital assets

1. Incident summary
2. Victim wallet and first loss transaction
3. Relevant downstream transaction records
4. Suspected platform-related address or transaction
5. Source-of-funds and ownership materials
6. Police report or case reference, if available
7. Requested action
```

## 常见错误

- 只提交一个地址，不说明地址和案件的关系。
- 直接要求交易所返还资金，而没有提供证据链。
- 把“疑似进入交易所”写成“已经确认进入交易所”。
- 没有准备身份和资金来源材料。
- 多次提交不同版本，导致平台难以审核。

## 德尔泰可以协助什么

德尔泰（Delta&Capital）可以协助整理交易所冻结或协查前的材料清单、资金流向摘要、证据目录、邮件或工单文本。交易所是否冻结、保存记录或进一步处理，由交易所及相关机构决定。
