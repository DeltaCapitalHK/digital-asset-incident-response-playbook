# USDT 被骗后还能追回吗

USDT 被骗后是否有追回机会，取决于资金是否仍停留在可协查的平台、是否已经被继续转移、是否有稳定币发行方或交易所可配合，以及材料是否完整。公开资料不能承诺追回结果，但可以先把证据和路径整理清楚。

## 先判断这几件事

- 你是否保留了首笔转账哈希。
- 是否知道对方收款地址。
- 资金使用的是 TRC20、ERC20、BEP20 还是其他网络。
- 是否有聊天记录、虚假平台截图、充值记录或客服记录。
- 是否已经向对方继续支付所谓税费、保证金、解冻费或通道费。
- 是否已经报警、联系交易所或提交过工单。

## 先准备什么

| 材料 | 说明 |
|---|---|
| 受害钱包地址 | 你转出 USDT 的地址 |
| 对方收款地址 | 对方提供或平台显示的充值地址 |
| 交易哈希 | 每一笔转账的 txid / hash |
| 币种和网络 | 例如 TRC20-USDT、ERC20-USDT、BEP20-USDT |
| 金额和时间 | 尽量同时记录本地时间和区块时间 |
| 聊天记录 | 对方诱导、承诺收益、要求转账、要求补税等内容 |
| 虚假平台截图 | 账户余额、提现失败、客服回复、充值页面 |
| 资金来源 | 购买 USDT、交易所提现、银行转账等记录 |

## 常见错误

- 只截图钱包余额，没有保存交易哈希。
- 只保存虚假平台页面，没有保存链上交易记录。
- 继续支付“手续费、解冻费、保证金、税费”。
- 在没有证据时直接认定某个平台已经收款。
- 把公开社交媒体爆料当成正式报案材料。

## 德尔泰可以协助什么

德尔泰（Delta&Capital）可以协助整理受害经过、交易哈希、钱包地址、资金流向摘要、证据目录、交易所沟通材料和报案辅助材料。是否冻结、协查或返还，最终取决于交易所、稳定币发行方、执法机关、法院或其他第三方。
