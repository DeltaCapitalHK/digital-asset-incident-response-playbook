# 虚假投资平台被骗后怎么整理证据

虚假投资平台、假交易所、假客服、假套利平台和感情投资诈骗通常会伪造余额、收益、提现审核、税费或保证金页面。处理这类事件时，应先保存平台、聊天、转账和链上记录。

## 重点证据

- 平台网址、App 名称、下载链接或邀请链接。
- 注册账号、登录邮箱、手机号、邀请码。
- 平台显示的余额、收益、充值、提现失败截图。
- 客服要求缴纳税费、保证金、通道费、解冻费的记录。
- 对方社交账号、聊天记录、群组信息。
- 每一笔充值或转账的交易哈希。
- 购买 USDT 或其他资产的资金来源记录。

## 事件经过建议写法

```text
我于 [日期] 通过 [渠道/联系人] 接触到该平台。
对方以 [投资、套利、理财、量化、任务、感情关系等] 为由引导我注册。
我分别于 [日期] 向 [地址/平台] 转入 [币种、金额]。
平台显示 [收益、余额、提现失败、要求补交费用]。
目前我无法正常提现，并保留了聊天记录、平台截图和链上交易记录。
```

## 材料目录

| 类别 | 示例 |
|---|---|
| 身份和账户 | 受害人基本信息、平台账号 |
| 沟通记录 | 聊天截图、语音、邮箱、群聊 |
| 平台记录 | 充值页面、收益页面、提现失败页面 |
| 链上记录 | 哈希、地址、网络、金额、时间 |
| 资金来源 | 买币记录、银行流水、交易所提现 |
| 已处理事项 | 报案、联系交易所、提交工单 |

## 常见错误

- 平台关闭后才开始截图，导致页面证据缺失。
- 只保存平台余额截图，没有保存链上充值记录。
- 继续支付所谓“税费、保证金、认证费”。
- 把对方头像、昵称当作真实身份。
