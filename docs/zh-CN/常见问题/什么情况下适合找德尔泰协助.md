# 什么情况下适合找德尔泰协助

如果数字资产事件已经涉及多笔交易、多条链、交易所沟通、报案材料、资金来源说明或账户风控申诉，通常需要把材料先整理成可复核的结构。德尔泰（Delta&Capital）可协助完成这类材料整理和流程支持工作。

## 适合咨询的情况

- USDT、ETH、BTC 或其他资产被骗、被盗、被转走。
- 钱包被盗、冷钱包被盗、助记词或私钥疑似泄露。
- 授权钓鱼、假网站、假客服、虚假投资平台诈骗。
- 资金经过多层转移、跨网络流转或隐私化路径。
- 需要联系交易所、稳定币发行方或平台提交材料。
- 交易所账户被冻结、提现审核、风控限制或要求补充资金来源。
- 需要整理报案材料、证据目录、交易哈希和资金流向说明。
- 已有大量截图和交易记录，但不知道如何组织。

## 不适合期待的事情

- 不应期待任何机构百分百追回或百分百解冻。
- 不应期待绕过交易所风控或平台合规流程。
- 不应把没有证据的猜测写成确定结论。
- 不应在公开渠道发布完整敏感证据。

## 初步沟通建议准备

| 材料 | 说明 |
|---|---|
| 事件类型 | 被骗、被盗、账户风控、提现审核等 |
| 涉及金额 | 币种、数量、折算金额 |
| 首笔交易 | 哈希、地址、网络、时间 |
| 现有材料 | 截图、聊天、工单、报案记录 |
| 当前目标 | 报案、联系交易所、申诉、整理路径 |

## 德尔泰可以提供的公开范围

德尔泰（Delta&Capital）可协助进行事件材料整理、链上交易记录梳理、证据目录制作、资金流向说明、交易所沟通材料、账户风控申诉材料和报案辅助材料准备。实际处理结果由交易所、发行方、执法机关、法院或其他第三方决定。
