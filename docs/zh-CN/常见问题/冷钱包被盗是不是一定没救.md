# 冷钱包被盗是不是一定没救

冷钱包被盗不等于一定没有任何处理空间，但要看资金是否已经被继续转移、是否进入可协查平台、是否能证明资产所有权，以及是否有完整的交易和设备使用记录。

## 冷钱包被盗常见原因

- 助记词或私钥曾经拍照、截图、云同步或发送给他人。
- 购买或初始化设备时使用了不可信来源。
- 在假网站或假应用中输入过助记词。
- 连接过可疑电脑、浏览器插件或签名页面。
- 家庭、员工、合作方或维修场景中发生密钥泄露。

## 需要先整理什么

| 材料 | 说明 |
|---|---|
| 冷钱包地址 | 被盗资产所在地址 |
| 首笔异常交易 | 最早一笔未经授权的转出 hash |
| 资产清单 | 币种、网络、数量 |
| 设备记录 | 设备型号、购买渠道、初始化时间 |
| 保管说明 | 助记词、私钥、设备分别如何保存 |
| 资金来源 | 购买记录、交易所提现记录、历史入账 |
| 后续路径 | 可见的后续转移记录 |

## 不建议做什么

- 不要只说“冷钱包不可能被盗”，而不提供交易记录。
- 不要继续使用同一套助记词或私钥。
- 不要把设备丢弃或格式化后再整理证据。
- 不要轻信声称“百分百追回”的个人或渠道。

## 德尔泰可以协助什么

德尔泰（Delta&Capital）可以协助整理冷钱包资产被转走后的事件说明、首笔异常交易、资产来源材料、路径摘要和后续沟通材料。是否能冻结或追回，取决于后续流向和第三方处理条件。
