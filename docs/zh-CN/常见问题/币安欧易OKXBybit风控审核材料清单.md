# 币安、欧易 OKX、Bybit 风控审核需要什么材料

不同交易所的审核要求不同，具体以平台官方通知、工单和邮件为准。本页只整理常见材料方向，适用于 Binance、OKX、Bybit 等中心化交易所账户被风控、提现审核、充值来源审查或要求补充说明的场景。

## 先确认平台要求

收到平台通知后，先确认：

- 是账户登录限制、提现限制、充值审查，还是全账户风控。
- 平台要求补充的是身份材料、资金来源、交易目的，还是某笔交易解释。
- 是否有工单号、邮件主题、截止时间和上传入口。
- 平台是否要求英文材料、中文材料或双语材料。

## 通用材料清单

| 材料 | 说明 |
|---|---|
| 账户信息 | UID、注册邮箱、手机号、工单号 |
| 身份证明 | 平台要求的身份证明、自拍验证或地址证明 |
| 资金来源 | 收入、经营、投资、借款、交易收益或其他合法来源说明 |
| 交易解释 | 相关充值、提现、换币、转账的目的和背景 |
| 链上记录 | 哈希、地址、网络、金额、时间 |
| 银行或交易所记录 | 入金、买币、提现、法币交易记录 |
| 对手方说明 | 如果涉及第三方收付款，需要说明关系和交易背景 |
| 辅助材料 | 合同、发票、工资流水、交易截图、聊天记录等 |

## 推荐整理格式

```text
Exchange:
UID:
Ticket ID:
Restricted function:
Date of restriction:
Requested documents:

Transaction summary:
- Date:
- Asset:
- Amount:
- Network:
- Tx hash:
- Address:
- Purpose:
- Source of funds:

Attached documents:
1.
2.
3.
```

## 不建议提交

- 前后不一致的资金来源说明。
- 无法解释来源的第三方收款记录。
- 大量无关截图。
- 非官方渠道要求提交的账户密码、验证码或完整敏感信息。
- 夸大事实、伪造材料或让他人代写虚假说明。
