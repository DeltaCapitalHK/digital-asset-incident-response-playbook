# 报警前需要准备哪些材料

数字资产被骗或被盗后，报警前应尽量把身份信息、受害经过、交易记录、聊天记录、平台信息和资金来源材料整理清楚。不同地区受理要求不同，本页只提供材料整理方向。

## 基础材料

- 受害人身份证明和联系方式。
- 事件发生时间、地点、经过。
- 对方联系方式、社交账号、群组、网址或 App 信息。
- 虚假平台页面、客服记录、提现失败记录。
- 转账交易哈希、钱包地址、币种、金额、时间。
- 购买数字资产或资金来源记录。
- 已联系交易所、平台或客服的记录。

## 事件经过建议写法

```text
我于 [日期] 通过 [渠道] 认识对方/接触平台。
对方以 [理由] 引导我转账。
我于 [日期] 分 [次数] 转出 [币种和金额]。
目前平台无法提现/对方失联/继续要求缴费。
我已保存聊天记录、平台截图、转账哈希和资金来源记录。
```

## 交易材料建议

| 项目 | 说明 |
|---|---|
| 转出平台 | 例如某交易所、钱包或银行卡买币渠道 |
| 转出钱包 | 受害钱包地址 |
| 收款地址 | 对方地址或平台充值地址 |
| 交易哈希 | 每笔链上交易 hash |
| 网络 | TRON、Ethereum、BNB Chain 等 |
| 金额 | 币种数量和折算金额 |
| 时间 | 本地时间和链上时间 |

## 常见错误

- 只带聊天截图，不带链上交易哈希。
- 只说“被骗了多少钱”，不能说明币种、网络和转账地址。
- 把平台余额截图当作真实资产证明。
- 报案材料前后时间线混乱。
