# 交易所账户被冻结怎么准备材料

交易所账户被冻结、提现审核、风控限制或要求补充材料时，重点不是情绪化申诉，而是把账户来源、交易目的、资金来源、链上路径和身份材料整理成平台可以审核的结构。

## 常见情况

- 账户登录或提现被限制。
- 提现进入人工审核。
- 平台要求说明资金来源。
- 平台要求说明某笔充值或提现的交易背景。
- 账户因为异常登录、交易对手、链上风险或合规审查被限制。
- 平台要求补充身份证明、地址证明、收入证明或交易说明。

## 材料清单

| 类型 | 需要准备 |
|---|---|
| 账户信息 | 平台名称、UID、注册邮箱或手机号、工单号 |
| 身份材料 | 平台要求的 KYC、地址证明或其他文件 |
| 资金来源 | 工资、经营收入、投资赎回、交易所提现、链上收款等说明 |
| 交易记录 | 涉及的充值、提现、交易、转账记录 |
| 链上记录 | 哈希、地址、网络、金额、时间 |
| 用途说明 | 充值、换币、提现、自用、经营或投资用途 |
| 沟通记录 | 平台邮件、客服聊天、工单回复 |

## 申诉说明建议结构

```text
1. 账户基本信息
2. 被限制的时间和功能
3. 涉及交易列表
4. 资金来源说明
5. 交易目的说明
6. 可提供的证明材料目录
7. 请求平台复核的事项
```

## 常见错误

- 只写“请帮我解冻”，不说明资金来源和交易背景。
- 多次提交前后矛盾的解释。
- 用截图代替交易哈希和账户记录。
- 隐瞒关键交易，导致后续审核更困难。
- 把申诉写成攻击平台或威胁客服。
