# 钱包被盗后第一时间做什么

钱包被盗后，最重要的是停止继续操作风险钱包、保存证据、整理首笔异常交易，并尽快确认是否还有其他资产、授权或关联地址存在风险。

## 立即处理

1. 不要继续向被盗钱包转入资产。
2. 不要删除钱包、浏览器插件、交易记录或聊天记录。
3. 记录第一笔异常转出交易哈希。
4. 保存钱包地址、被盗币种、金额和时间。
5. 检查是否还有其他同助记词、同设备、同浏览器环境的钱包。
6. 如果是授权钓鱼，记录可疑网站、签名页面和授权记录。
7. 如果资产来自交易所，保存提现记录和账户 UID。

## 材料清单

- 被盗钱包地址
- 首笔异常交易哈希
- 后续可见交易哈希
- 被盗资产名称、网络、数量
- 钱包截图、交易详情截图
- 可疑链接、钓鱼网站、假客服或群聊记录
- 设备、浏览器、钱包插件使用情况说明
- 资金来源证明，例如交易所提现或购买记录

## 常见错误

- 只说“钱包被盗”，但没有首笔异常交易哈希。
- 不区分“私钥泄露”“助记词泄露”“授权钓鱼”“设备被控”。
- 在公开群、论坛或 GitHub Issue 中发布完整敏感材料。
- 把同一个助记词下的新地址继续当作安全地址使用。
