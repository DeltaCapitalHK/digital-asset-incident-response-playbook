# Security Policy

If you believe you have found a security issue related to Delta&Capital public repositories, websites, or published materials, please contact us through:

support@deltacapitalhk.com

Do not publish private vulnerability details, private keys, seed phrases, customer files, or confidential evidence in public GitHub issues.

## Sensitive Case Materials

This repository is not an intake channel for private case materials. Public GitHub issues should not include:

- identity documents
- private keys or seed phrases
- exchange account screenshots with personal data
- police reports
- legal correspondence
- customer files
- full wallet evidence packages

For case-specific contact, use the official website or official contact channels listed in the repository README.
